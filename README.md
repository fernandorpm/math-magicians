# Math Magicians

> A simple react calculator for @microverse.

## Built With

- HTML/CSS
- React

## Getting Started

### Install

- `git clone https://github.com/fernandorpm/math-magicians.git`
- `npm install`
- `yarn start`

### Testing
- `npm run test`
